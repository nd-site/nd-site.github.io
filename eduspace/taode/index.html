Hướng Dẫn Viết File data.js

File cần khai báo `window.quizData`, chứa thông tin đề, cấu hình thời gian/điểm và danh sách câu hỏi.

EduSpace Schema

Cấu trúc bắt buộc

Dùng `window.quizData = {...};`. Không dùng JSON thuần. `testDuration` tính bằng giây. `examLayout` quy định số câu và điểm mỗi dạng.

Các dạng câu

`multiple`: trắc nghiệm, `correct` là index 0=A. `truefalse`: mảng phát biểu và mảng `correctAnswers`. `short`: đáp án ngắn. `essay`: tự luận.

Nội dung nâng cao

Có thể dùng Markdown, LaTeX với dấu gạch chéo kép trong chuỗi JS, hình ảnh `![mô tả](url)`, âm thanh `[audio:url]`, video `[video:url]`.

window.quizData = {
  "title": "Đề kiểm tra mẫu - EduSpace by ND Labs",
  "pageTitle": "Đề kiểm tra mẫu",
  "examInfo": {
    "subject": "Tên môn học",
    "grade": "Lớp 12",
    "date": "Năm học 2025 - 2026",
    "examName": "Kiểm tra Cuối Học Kì",
    "attempt": "Bản 1"
  },
  config: {
    testDuration: 2700,
    examLayout: {
      multiple: { count: 1, pointsPerQ: 0.25 },
      truefalse: { count: 1, pointsPerQ: 1, partialScoring: [0.1, 0.25, 0.5, 1] },
      short: { count: 1, pointsPerQ: 0.5 },
      essay: { count: 0, pointsPerQ: 0 }
    }
  },
  questions: [
    {
      type: "multiple",
      question: "1 + 1 bằng mấy?",
      options: ["1", "2", "3", "4"],
      correct: 1,
      explanation: "1 + 1 = 2."
    },
    {
      type: "truefalse",
      question: "Xét các phát biểu sau:",
      options: ["HTML là ngôn ngữ đánh dấu.", "CSS dùng để truy vấn dữ liệu."],
      correctAnswers: [true, false],
      correct: [true, false]
    },
    {
      type: "short",
      question: "Kết quả của $2^3$ là bao nhiêu?",
      correct: "8"
    }
  ]
};

Bảng thuộc tính nhanh

`title`: tên đề hiển thị ở màn hình bắt đầu.

`config.testDuration`: thời gian làm bài, tính bằng giây.

`config.examLayout[type].count`: số câu lấy theo dạng khi chọn làm theo ma trận.

`config.examLayout[type].pointsPerQ`: điểm mỗi câu của dạng đó.

`questions`: ngân hàng câu hỏi, hệ thống lấy theo thứ tự trong mảng.

Lỗi thường gặp cần tránh

Thiếu dấu phẩy giữa các câu hỏi hoặc phương án.

Dùng `correct: 4` cho câu có 4 đáp án. Index đúng phải là 0, 1, 2, 3.

Viết công thức LaTeX bằng một dấu `\`; trong chuỗi JavaScript cần dùng `\\`.

Dạng Đúng/Sai thiếu `correctAnswers` hoặc số đáp án không khớp số phát biểu.

Quy tắc làm đề chuẩn EduSpace

Yêu cầu bắt buộc để AI duyệt và cho phép xuất file

Nội dung chuyên môn

  • Kiến thức phải chính xác tuyệt đối, không có sai sót khoa học.
  • Nội dung bám sát chương trình GDPT mới nhất của Bộ GD&ĐT.
  • Câu hỏi rõ ràng, không mập mờ, không gây hiểu lầm.
  • Đáp án chọn làm "đúng" phải là đáp án duy nhất đúng/chuẩn nhất.

Hình thức & Trình bày

  • Viết hoa đầu câu và kết thúc bằng dấu chấm (nếu là câu khẳng định).
  • Dấu câu đặt sát ký tự trước, cách ký tự sau (Ví dụ: "sai ,đúng" -> "sai, đúng").
  • Định dạng LaTeX bọc trong dấu $: $E=mc^2$.
  • Tên riêng, thuật ngữ khoa học phải viết đúng chuẩn.

Cấu trúc kỹ thuật

  • Trắc nghiệm (Dạng 1): Luôn có 4 phương án, 1 đáp án đúng.
  • Đúng/Sai (Dạng 2): Chứa 4 lệnh phát biểu con a, b, c, d.
  • Trả lời ngắn (Dạng 3): Đáp án là số hoặc từ khóa cực ngắn (max 4 ký tự).
  • Tự luận (Dạng 4): Phải có hướng dẫn chấm chi tiết trong phần giải thích.

Tiêu chí chấm điểm của AI:

AI sẽ chấm điểm dựa trên thang 100%. Đề thi cần đạt tối thiểu 75% điểm chất lượng mới được phép xuất file. AI sẽ soi từng lỗi chính tả, ngữ pháp và tính logic của kiến thức.
