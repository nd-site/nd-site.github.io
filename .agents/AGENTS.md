# Project Rules

- Khi viết code, sửa code bao gồm tính năng mới, sửa tính năng, ... (nhưng không bao gồm thêm bài kiểm tra mới trong eduspace), bạn phải luôn tự động cập nhật số phiên bản trong [version.js](file:///d:/Project/WebSite/EduSpace%20by%20ND%20Labs/assets/js/version.js).
- Định dạng phiên bản: `ver:<năm (2025=0, 2026=1)>.<tháng>.<ngày>.<giờ><phút>`
  Ví dụ với ngày 22/07/2026 lúc 23:51 -> `ver:1.7.22.2351`
